# Help me choose

A WordPress plugin that helps customers choose the right product or service through an interactive decision tree.

## Description

This plugin provides a multi-step decision tree form that guides users through a series of questions to help them find the most suitable product or service. It features:

- **Multi-step decision tree** with conditional branching
- **AI-powered configuration wizard** for easy setup
- **Customizable styling** with preset styles
- **Recommendation system** with support for pages, posts, products, and custom post types
- **Usage tracking** and statistics
- **Responsive design** for mobile and desktop
- **Translation ready** (Finnish included)

## Installation

1. Upload the plugin files to `/wp-content/plugins/auta-minua-valitsemaan/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Tools → Help me choose to configure

## Configuration

### Quick Setup (Installation Wizard)

1. Go to Tools → Help me choose → Installation Wizard
2. Generate a starter pack JSON
3. Upload it to the AI configuration generator
4. Import the generated configuration
5. Customize as needed

### Manual Setup

1. Go to Tools → Help me choose → Steps
2. Add steps and configure options
3. Set target steps for conditional navigation
4. Configure recommendations in the Recommendations tab
5. Customize styling in the Styling tab

## Features

- **Conditional Navigation**: Each option can target a specific step or jump to recommendations
- **Multiple Recommendations**: Support for multiple recommendation bundles
- **Rich Content**: Display pages, posts, products, and custom post types as cards
- **Image Support**: Add images to options
- **Style Presets**: Choose from preset styles or customize everything
- **Usage Tracking**: Track user interactions and completion rates
- **Import/Export**: Export and import configurations as JSON

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- WooCommerce (optional, for product support)
